# ArtifyMotion

这是一个对GIF和图片进行二次创作的平台，可以对GIF和图片进行人脸替换、文字添加等操作。平台提供一系列的GIF和图片以及可替换的素材，同时也可以由用户自主上传GIF及替换素材。

## 硬件

* **OS**: CentOS 7x64
* **CPU**: 4核心8线程
* **内存**：8G
* **带宽**：10M

## 开发环境

* 服务器：Nginx
* 后端：Django, MySQL8.0
* 前端：Vue3, npm@9.7.2, node@v20.4.0
